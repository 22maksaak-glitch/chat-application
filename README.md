# 💬 Chat App – Мультиязычный чат с комнатами

Реализация простого веб-чата с поддержкой нескольких комнат, ников и истории сообщений. Демонстрирует работу **WebSockets** в реальном времени на трёх популярных бэкендах:

- **Node.js** – Express + Socket.IO
- **Python** – FastAPI + WebSockets
- **Go** – Gorilla WebSocket

Каждая версия полностью самодостаточна, использует единый фронтенд (HTML/CSS/JS) и одинаковый API событий.

## 🚀 Ключевые возможности

- ✅ Создание / подключение к произвольной комнате
- ✅ Обмен сообщениями в реальном времени (WebSocket)
- ✅ Отображение списка активных пользователей в комнате
- ✅ Сохранение последних 50 сообщений (в памяти)
- ✅ Простая авторизация по никнейму
- ✅ Адаптивный интерфейс (тёмная тема)

## 🧠 Почему средняя сложность?

- Асинхронная обработка WebSocket-соединений
- Управление состоянием (комнаты, пользователи, история)
- Broadcast сообщений всем участникам комнаты
- Возможность масштабирования через Redis (опционально, не реализовано)
